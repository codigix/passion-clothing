## â“ FAQ

**Q: Can I upload more than 5 files?**  
A: No, maximum is 5 files per order to keep UI clean and loading fast.

**Q: What if I exceed 5MB per file?**  
A: The system will show an error. Compress the file or split into multiple files.

**Q: Can I replace a file without removing and re-uploading?**  
A: Not directly - remove the old one and upload new one in its place.

**Q: Are files stored on the server?**  
A: Currently, file names are stored. Full file upload features coming soon.

**Q: Can I download all patterns as ZIP?**  
A: Not yet, but this feature is planned for future release.

**Q: What if I don't upload any patterns?**  
A: That's fine - patterns are optional. Order will be created normally.

**Q: Do patterns affect order pricing?**  
A: No, patterns are informational only. They don't change the price.
