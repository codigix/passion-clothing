### Slide-up Animation
```javascript
@keyframes slideUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
.animate-slideUp { animation: slideUp 0.4s ease-out; }
// Applied to: Form sections, help box
```

  }
  to {
    opacity: 1;

}
